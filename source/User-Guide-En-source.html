How to play

  • Simply press one of the Play buttons at the top left in the title bar or at the bottom at the end of the random draw list, and the new draws will be displayed.
  • Another convenient way to play is to briefly shake the device so new draws are displayed automatically.
  • You can also pull down to refresh the draw list with new random draws.
  • Or you can click ont the Game icone at the bottom of the Alea screen to trigger a new set of random draws.
  • By clicking on Edit you can, if you wish, de-select the draws that do not interest you.
  • And conversely, you can re-select the draws you want if necessary.
  • To make the selection easier, a None - All selector allows you to select or unselect all the draws.
  • In this Edit screen you can also reorder the random draws in any order you like.
  • By default, each time you change screen (Edit, Parameter or Help), a new random draw is made without you needing to press any Play or Game button. Unless you modify this option : see below Setting the random draws display.

Random draw of a digit

  • By default the draw is between 1 and 10.
  • It is possible to specify the minimum and maximum number between which one wishes the random draw, in the Parameter screen.
  • Minimum and maximum are included. The maximum number must be greater than the minimum number.
  • By default, a number is drawn only once. For example, the number 3 will only appear once if the Repetition number is set to No (default position). When all the numbers between the minimum and maximum (between 1 and 10 by default) have been drawn, then the draw starts again from zero and all the numbers can be drawn again (once only).
  • However, digit repetition is possible, simply select the Digit repetition to yes selector in the Parameter screen.
  • The random draw is for a single digit by default.
  • It is however possible to increase the number of digits randomly drawn up to 5 in the Parameter screen.

Random draw of a letter

  • Of course, a letter is drawn between 'A' and 'Z'.
  • By default, a letter is only drawn once. For example the letter 'P' will only appear once if the Repetition letter is set to No (default position). When all letters have been drawn, then the random draw starts from scratch and all letters can be drawn again (once only).
  • However, it is possible to repeat a letter, simply select the Letter repetition selector to Yes in the Parameter screen.
  • The random draw is for a single letter by default.
  • It is however possible to increase the number of letters randomly drawn up to 5 in the Parameter screen.

Random draw of a dice

  • A dice number can appear as many times as chance allows.
  • By default the draw is for one single dice.
  • It is however possible to increase the number of dices drawn up to 5 in the Parameter screen.

top↑

Random draw of a card

  • A playing card is drawn only once : when a card has been drawn it will no longer appear.
  • When all 52 cards deck has been drawn (52 card draws), then the cards deck is reset and shuffled. And you can start again.
  • By default the draw is for a single card.
  • It is however possible to increase the number of cards randomly drawn up to 5 in the Parameter screen.

Draw for the roulette wheel

  • The casino roulette returns a number between 0 and 36.
  • The same number can come up as many times as chance would have it.
  • At each random draw, the number is displayed with its characteristics:
    • even or odd
    • red or black (and green for zero).
    • in the first 12 numbers from 1 to 12 (1st12), in the middle 12 numbers from 13 to 24 (2nd12) or in the last 12 numbers from 25 to 36 (3rd12)
    • in the first column (C1), in the 2nd column (C2) or in the 3rd column (C3).

Random draw of one day

  • The day of the week appears only once.
  • When all the days are drawn, the random draw starts a new week.
  • Day repetition is however possible, just select the Day repetition selector to yes in the Parameter screen.

Random draw for Euromillion

  • 5 digits are drawn at random between 1 and 50.
  • And 2 “star” numbers are drawn at random between 1 and 12.
  • Each random draw is independent, the same number can be issued in several successive draws.
  • Please note: statistics relate only to the 5 main digits and not to the 2 “star” digits.

Random draw for Lottery

  • 5 digits are drawn at random between 1 and 49.
  • And 1 number “chance” is drawn at random between 1 and 10.
  • Each random draw is independent, the same number can be issued in several successive draws.
  • Please note: statistics relate only to the 5 main figures and not to the number “chance”.

Historic of one random draw

  • To consult the historic of a given random draw, simply click on this draw in the screen Alea.
  • The number of random draws historized is limited to 1,000 so as not to overload your device's memory.
  • A Delete button at the top right allows you to delete the historic of this draw if you wish.
  • Note: the button Reset in the screen Parameter allows to delete all the historics of all the random draws.

top↑

Statistics of one random draw

  • To know the statistics of a given random draw, you can click on the button Statistics at the bottom of the screen Historic.
  • Statistics are independent from the historic. For example, you can reset the historic of a random draw without affecting the statistics. And vice versa.
  • A Delete button at the top right allows you to delete the statistics of this draw if you wish.
  • Note: the button Reset in the screen Parameter allows to delete all the statistics of all the random draws.

top↑

Reset all random draws

  • The Reset button in the Parameter screen returns you to the default values.
  • It also resets all random draws: the digits, letters, cards and days in particular are reset to zero. And you can start over as if there's never been a draw.
  • The selection of random draws and their order are also reset to their default values.
  • The historics as well as the statistics of all random draws are reset to zero.
  • Apart from the reset, the settings are saved from one session to another.

Setting the random draws display

  • It is possible to display the Remaining draws by setting the selector to yes in the parameter screen (the selector is no by default). Thus, for example, the number of cards remaining in the deck will be displayed at each draw. Same for the number of digits, letters and days remaining to be drawn.
  • By default, an automatic draw is performed each time the Alea draws screen is displayed (when changing screens) or when you click on the Game icon at the bottom of the screen. It is however possible to disable this automatism by setting the selector Automatic draws to no in the parameter screen. In this case, the random draws will only be done when you manually trigger them with the Play buttons or by pulling down the list of draws.
